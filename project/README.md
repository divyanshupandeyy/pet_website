## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/smart-finance.git
   ```

2. Install dependencies:
   ```bash
   cd smart-finance
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
